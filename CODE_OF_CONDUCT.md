# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

The full text is published at the link above and applies to all project spaces — issues, pull requests, discussions, code review, and any other forum where contributors interact about this project.

## Reporting

To report a concern, contact the maintainer privately. Reports are handled confidentially.

- Email: contact@myika.ai
- For urgent or sensitive matters, you may also use a [private GitHub security advisory](https://github.com/Myikaai/myika-unreal/security/advisories/new).

Reporters will receive an acknowledgment within 7 days. Maintainers commit to investigating reports in good faith and to applying the enforcement guidelines published in the Contributor Covenant.
